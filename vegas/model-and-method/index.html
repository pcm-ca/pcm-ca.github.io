Monte Carlo Metropolis algorithm

ùïçegas simulates the time evolution of a classical spin magnetic system using the Monte Carlo Metropolis algorithm, where new configurations of a system are generated from a previous state using a transition probability. Different spin update policies are implemented in ùïçegas, including a highly efficient adaptive move.
